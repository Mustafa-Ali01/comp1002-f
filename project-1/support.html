Need Assistance? We're Here to Help.

Universal Communications is committed to 24/7 reliability. If you experience an issue with a Model 700 Phone Booth, please follow the steps below.

Frequently Asked Questions (FAQ)

Q: My dime was taken but the call didn't connect. What should I do?

A: Our automatic coin return should handle this. If it failed, please call the repair number (below) and provide the booth's location identifier. We will mail a refund check immediately.

Q: The interior light is out. Is the phone still operational?

A: Yes, the light is a separate circuit. The telephone itself should still be fully functional. Please report the booth location so our repair crew can replace the bulb.

Q: Can I use this booth for long-distance calls?

A: Absolutely! The Model 700 accepts all major calling cards for long-distance service, or you can call the operator for a collect call.
